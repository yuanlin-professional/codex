# codex-cli/scripts - 构建与部署脚本

## 功能概述

此目录包含 Codex CLI 的构建、打包和部署辅助脚本。

## 目录结构

| 文件 | 说明 |
|------|------|
| `build_container.sh` | 构建 Docker 容器镜像 |
| `build_npm_package.py` | 构建 npm 发布包 |
| `init_firewall.sh` | 初始化防火墙规则 |
| `install_native_deps.py` | 安装原生依赖 |
| `run_in_container.sh` | 在容器内运行 Codex |
| `README.md` | 脚本使用说明 |
