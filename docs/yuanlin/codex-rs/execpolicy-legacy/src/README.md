# execpolicy-legacy/src — 源代码目录

## 功能概述
execpolicy-legacy crate 的源代码实现目录。遗留的命令执行策略引擎。

## 目录结构
| 文件 | 说明 |
|------|------|
| `arg_matcher.rs` | 参数匹配器 |
| `arg_resolver.rs` | 参数解析器 |
| `arg_type.rs` | 参数类型 |
| `default.policy` | 默认策略文件 |
| `error.rs` | 错误类型 |
| `exec_call.rs` | 执行调用 |
| `execv_checker.rs` | execv 检查器 |
| `lib.rs` | 库入口 |
| `main.rs` | 程序入口 |
| `opt.rs` | 命令选项 |
| `policy_parser.rs` | 策略解析器 |
| `policy.rs` | 策略定义 |
| `program.rs` | 程序定义 |
| `sed_command.rs` | sed 命令处理 |
| `valid_exec.rs` | 合法执行验证 |
