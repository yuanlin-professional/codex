# templates/agents/ -- 多代理编排模板

## 功能概述

`agents/` 包含多代理系统中编排器（orchestrator）的提示词模板。该模板是一份完整的系统提示词，定义了 AI 代理作为 "Codex" 协作型编程助手的核心人格、语调风格规范、响应准则、文件引用格式以及多代理协调使用指导。

## 目录结构

```
agents/
└── orchestrator.md    # 编排器指令模板
```

## 依赖关系

- **被引用方**：`src/models_manager/model_info.rs`（构建基础指令时引用编排器模板）

## 核心接口/API

| 文件 | 说明 |
|---|---|
| `orchestrator.md` | 编排器系统提示词，包含：人格定义（协作型结对编程 AI）、语调规范（简洁直接友好）、响应风格（命令行 UI 优化、GitHub Markdown）、文件引用格式规范（绝对路径/相对路径/行号引用）、多代理使用最佳实践 |
