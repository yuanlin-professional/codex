# templates/collab/ -- 协作模式实验性提示词

## 功能概述

`collab/` 包含 Codex 协作模式的实验性提示词模板。当前提供多代理协作的指导性提示，指导模型如何有效地生成和管理子代理来完成复杂任务，包括适用场景判断和使用注意事项。

## 目录结构

```
collab/
└── experimental_prompt.md    # 实验性多代理协作提示词
```

## 依赖关系

- **被引用方**：协作模式系统在启用实验性多代理功能时引用此模板

## 核心接口/API

| 文件 | 说明 |
|---|---|
| `experimental_prompt.md` | 多代理协作指导，涵盖：适用场景（大型多范围任务、代码审查、辩论讨论、测试执行与修复）、环境隔离注意事项、上下文优化建议、超时参数设置、子代理工具权限管理 |
