# `mod.rs` -- 模型管理器模块入口

## 功能概述

模型管理器的模块入口文件，声明子模块并提供 `client_version_to_whole` 工具函数，用于将 Cargo 包版本号转换为主版本号格式（如 "1.2.3-alpha.4" 变为 "1.2.3"），供缓存版本校验使用。

## 关键函数/方法

| 函数 | 签名摘要 | 说明 |
|------|----------|------|
| `client_version_to_whole` | `fn() -> String` | 从编译时环境变量提取主版本号，格式为 `MAJOR.MINOR.PATCH` |

## 依赖关系

- **引用的 crate**: 无外部依赖，使用 `env!` 宏读取编译时版本
- **被引用方**: `manager.rs` 中用于缓存版本校验
