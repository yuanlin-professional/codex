# `mod.rs` — 状态管理模块入口

## 功能概述

`state` 模块的入口文件，组织和重导出 Codex 会话状态管理相关的子模块。包括会话服务层、会话状态和活动轮次管理三个子模块，提供会话生命周期中所需的核心状态类型。

## 关键结构体/枚举

| 名称 | 类型 | 说明 |
|------|------|------|
| `SessionServices` | 结构体（重导出） | 会话级服务依赖（如 HTTP 客户端、数据库连接等） |
| `SessionState` | 结构体（重导出） | 会话全局状态，维护配置和运行时上下文 |
| `ActiveTurn` | 结构体（重导出） | 当前活动轮次的状态 |
| `RunningTask` | 结构体（重导出） | 正在执行的任务实例 |
| `TaskKind` | 枚举（重导出） | 任务类型（如 shell 命令、apply_patch 等） |

## 依赖关系

- **引用的 crate**: 无直接外部依赖（通过子模块引用）
- **被引用方**: `codex-core` 的会话循环和工具执行逻辑
