# execpolicy/src — 源代码目录

## 功能概述
execpolicy crate 的源代码实现目录。命令执行策略引擎。

## 目录结构
| 文件 | 说明 |
|------|------|
| `amend.rs` | 策略修正 |
| `decision.rs` | 决策逻辑 |
| `error.rs` | 错误类型 |
| `execpolicycheck.rs` | 策略检查 |
| `executable_name.rs` | 可执行文件名 |
| `lib.rs` | 库入口 |
| `main.rs` | 程序入口 |
| `parser.rs` | 策略解析器 |
| `policy.rs` | 策略定义 |
| `rule.rs` | 规则定义 |
