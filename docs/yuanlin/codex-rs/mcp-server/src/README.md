# mcp-server/src — 源代码目录

## 功能概述
mcp-server crate 的源代码实现目录。实现 MCP (Model Context Protocol) 服务器。

## 目录结构
| 文件 | 说明 |
|------|------|
| `codex_tool_config.rs` | Codex 工具配置 |
| `codex_tool_runner.rs` | Codex 工具运行器 |
| `exec_approval.rs` | 执行审批 |
| `lib.rs` | 库入口 |
| `main.rs` | 程序入口 |
| `message_processor.rs` | 消息处理器 |
| `outgoing_message.rs` | 发出消息 |
| `patch_approval.rs` | 补丁审批 |
