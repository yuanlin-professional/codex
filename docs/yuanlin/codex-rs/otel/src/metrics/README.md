# otel/src/metrics — 指标模块

## 功能概述
OpenTelemetry 指标收集和报告模块。

## 目录结构
| 文件 | 说明 |
|------|------|
| `client.rs` | 指标客户端 |
| `config.rs` | 指标配置 |
| `error.rs` | 错误类型 |
| `mod.rs` | 模块入口 |
| `names.rs` | 指标名称 |
| `runtime_metrics.rs` | 运行时指标 |
| `tags.rs` | 指标标签 |
| `timer.rs` | 计时器 |
| `validation.rs` | 指标验证 |
