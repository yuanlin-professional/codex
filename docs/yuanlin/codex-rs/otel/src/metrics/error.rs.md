# `error.rs` — 指标系统错误类型定义

## 功能概述
该文件定义了 `MetricsError` 枚举和 `Result<T>` 类型别名，涵盖指标系统中可能遇到的所有错误场景：空/非法指标名称、空/非法标签组件、导出器禁用、负数计数器增量、OTLP 导出器构建失败、Provider 关闭失败、以及运行时快照不可用等。

## 关键结构体/枚举
| 名称 | 类型 | 说明 |
|------|------|------|
| `MetricsError` | 枚举 | 指标系统错误类型，包含 11 个变体 |

## 关键函数/方法

|------|----------|------|

## 依赖关系
- **引用的 crate**: `thiserror`, `opentelemetry_otlp`, `opentelemetry_sdk`
- **被引用方**: 整个 `metrics` 子系统
