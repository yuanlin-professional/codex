# otel/tests/suite — 测试套件

## 功能概述
otel 的测试套件模块。

## 目录结构
| 文件 | 说明 |
|------|------|
| `manager_metrics.rs` | 指标管理器测试 |
| `mod.rs` | 模块入口 |
| `otel_export_routing_policy.rs` | 导出路由策略测试 |
| `otlp_http_loopback.rs` | OTLP HTTP 回环测试 |
| `runtime_summary.rs` | 运行时摘要测试 |
| `send.rs` | 发送测试 |
| `snapshot.rs` | 快照测试 |
| `timing.rs` | 计时测试 |
| `validation.rs` | 验证测试 |
