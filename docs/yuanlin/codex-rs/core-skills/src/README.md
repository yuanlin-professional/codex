# core-skills/src — 源代码目录

## 功能概述
core-skills crate 的源代码实现目录。管理内置技能的加载、注入和调用。

## 目录结构
| 文件 | 说明 |
|------|------|
| `config_rules.rs` | 配置规则 |
| `env_var_dependencies.rs` | 环境变量依赖 |
| `injection.rs` | 技能注入 |
| `invocation_utils.rs` | 调用工具函数 |
| `lib.rs` | 库入口 |
| `loader.rs` | 技能加载器 |
| `manager.rs` | 技能管理器 |
| `mention_counts.rs` | 提及计数 |
| `model.rs` | 技能模型 |
| `remote.rs` | 远程技能 |
| `render.rs` | 技能渲染 |
| `system.rs` | 系统技能 |
