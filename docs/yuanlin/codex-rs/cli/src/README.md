# cli/src — 源代码目录

## 功能概述
cli crate 的源代码实现目录。Codex 命令行界面主入口。

## 目录结构
| 文件 | 说明 |
|------|------|
| `app_cmd.rs` | 应用命令 |
| `debug_sandbox.rs` | 调试沙箱命令 |
| `exit_status.rs` | 退出状态码 |
| `lib.rs` | 库入口 |
| `login.rs` | 登录命令 |
| `main.rs` | 程序入口 |
| `mcp_cmd.rs` | MCP 命令 |
| `wsl_paths.rs` | WSL 路径处理 |
