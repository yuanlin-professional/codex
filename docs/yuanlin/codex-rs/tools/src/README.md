# tools/src — 源代码目录

## 功能概述
tools crate 的源代码实现目录。定义和管理各类工具（agent、MCP、本地工具等）。

## 目录结构
| 文件 | 说明 |
|------|------|
| `agent_job_tool.rs` | Agent 作业工具 |
| `agent_tool.rs` | Agent 工具 |
| `code_mode.rs` | 代码模式工具 |
| `dynamic_tool.rs` | 动态工具 |
| `js_repl_tool.rs` | JS REPL 工具 |
| `json_schema.rs` | JSON Schema 处理 |
| `lib.rs` | 库入口 |
| `local_tool.rs` | 本地工具 |
| `mcp_resource_tool.rs` | MCP 资源工具 |
| `mcp_tool.rs` | MCP 工具 |
| `request_user_input_tool.rs` | 请求用户输入工具 |
| `responses_api.rs` | Responses API 工具 |
| `tool_definition.rs` | 工具定义 |
| `tool_spec.rs` | 工具规格 |
| `utility_tool.rs` | 实用工具 |
