# rmcp-client/src — 源代码目录

## 功能概述
rmcp-client crate 的源代码实现目录。实现 RMCP (Remote MCP) 客户端协议。

## 目录结构
| 文件 | 说明 |
|------|------|
| `auth_status.rs` | 认证状态 |
| `lib.rs` | 库入口 |
| `logging_client_handler.rs` | 日志客户端处理器 |
| `oauth.rs` | OAuth 认证 |
| `perform_oauth_login.rs` | OAuth 登录流程 |
| `program_resolver.rs` | 程序解析器 |
| `rmcp_client.rs` | RMCP 客户端核心 |
| `utils.rs` | 工具函数 |
