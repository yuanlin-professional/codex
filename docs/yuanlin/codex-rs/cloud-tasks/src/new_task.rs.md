# `new_task.rs` — 新建任务页面状态

## 功能概述
定义 TUI 中"新建任务"页面的状态结构体 `NewTaskPage`，封装 Composer 输入组件、提交状态、环境 ID 和 best-of-N 尝试次数。提供快捷键提示的初始化配置和默认值实现。

## 关键结构体/枚举
| 名称 | 类型 | 说明 |
|------|------|------|
| `NewTaskPage` | 结构体 | 新建任务页面状态：composer 输入、是否提交中、环境 ID、尝试次数 |

## 依赖关系
- **引用的 crate**: `codex_tui`
- **被引用方**: `app.rs`
