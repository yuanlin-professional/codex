# cloud-tasks/src — 源代码目录

## 功能概述
cloud-tasks crate 的源代码实现目录。云任务的 TUI 界面和管理。

## 目录结构
| 文件 | 说明 |
|------|------|
| `app.rs` | 应用框架 |
| `cli.rs` | 命令行参数 |
| `env_detect.rs` | 环境检测 |
| `lib.rs` | 库入口 |
| `new_task.rs` | 创建新任务 |
| `scrollable_diff.rs` | 可滚动差异视图 |
| `ui.rs` | 用户界面 |
| `util.rs` | 工具函数 |
