# hooks/schema/generated — 生成的 Schema

## 功能概述
hooks 事件的自动生成 JSON Schema 文件。定义各钩子事件的输入输出格式。

## 目录结构
| 文件 | 说明 |
|------|------|
| `post-tool-use.command.input.schema.json` | 工具使用后事件输入 |
| `post-tool-use.command.output.schema.json` | 工具使用后事件输出 |
| `pre-tool-use.command.input.schema.json` | 工具使用前事件输入 |
| `pre-tool-use.command.output.schema.json` | 工具使用前事件输出 |
| `session-start.command.input.schema.json` | 会话开始事件输入 |
| `session-start.command.output.schema.json` | 会话开始事件输出 |
| `stop.command.input.schema.json` | 停止事件输入 |
| `stop.command.output.schema.json` | 停止事件输出 |
| `user-prompt-submit.command.input.schema.json` | 用户提示提交输入 |
| `user-prompt-submit.command.output.schema.json` | 用户提示提交输出 |
