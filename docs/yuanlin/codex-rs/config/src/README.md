# config/src — 源代码目录

## 功能概述
config crate 的源代码实现目录。管理 Codex 的配置系统。

## 目录结构
| 文件 | 说明 |
|------|------|
| `cloud_requirements.rs` | 云端需求配置 |
| `config_requirements.rs` | 配置需求 |
| `constraint.rs` | 约束定义 |
| `diagnostics.rs` | 诊断信息 |
| `fingerprint.rs` | 指纹计算 |
| `lib.rs` | 库入口 |
| `merge.rs` | 配置合并 |
| `overrides.rs` | 配置覆盖 |
| `project_root_markers.rs` | 项目根标记 |
| `requirements_exec_policy.rs` | 执行策略需求 |
| `skills_config.rs` | 技能配置 |
| `state.rs` | 配置状态 |
