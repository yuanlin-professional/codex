# codex-api/src/endpoint/realtime_websocket — 实时 WebSocket

## 功能概述
实时 WebSocket 端点实现，支持 V1 和 V2 协议。

## 目录结构
| 文件 | 说明 |
|------|------|
| `methods_common.rs` | 公共方法 |
| `methods_v1.rs` | V1 方法 |
| `methods_v2.rs` | V2 方法 |
| `methods.rs` | 方法入口 |
| `mod.rs` | 模块入口 |
| `protocol_common.rs` | 公共协议 |
| `protocol_v1.rs` | V1 协议 |
| `protocol_v2.rs` | V2 协议 |
| `protocol.rs` | 协议入口 |
