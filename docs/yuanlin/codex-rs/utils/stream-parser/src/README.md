# stream-parser/src — 源代码目录

## 功能概述
stream-parser 工具 crate 的源代码实现目录。解析流式文本输出。

## 目录结构
| 文件 | 说明 |
|------|------|
| `assistant_text.rs` | 助手文本解析 |
| `citation.rs` | 引用解析 |
| `inline_hidden_tag.rs` | 内联隐藏标签 |
| `lib.rs` | 库入口 |
| `proposed_plan.rs` | 建议计划解析 |
| `stream_text.rs` | 流式文本处理 |
| `tagged_line_parser.rs` | 标记行解析器 |
| `utf8_stream.rs` | UTF-8 流处理 |
