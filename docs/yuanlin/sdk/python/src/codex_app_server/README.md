# codex_app_server — Python 客户端包

## 功能概述
Codex 应用服务器的 Python SDK 客户端实现。

## 目录结构
| 文件 | 说明 |
|------|------|
| `__init__.py` | 包入口 |
| `_inputs.py` | 输入处理 |
| `_run.py` | 运行逻辑 |
| `api.py` | API 定义 |
| `async_client.py` | 异步客户端 |
| `client.py` | 同步客户端 |
| `errors.py` | 错误类型 |
| `models.py` | 数据模型 |
| `py.typed` | PEP 561 类型标记 |
| `retry.py` | 重试逻辑 |
