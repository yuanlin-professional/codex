# sdk/python/examples - SDK 使用示例

## 功能概述

包含 14 个 Python SDK 使用示例，每个示例提供同步（`sync.py`）和异步（`async.py`）两个版本。

## 目录结构

| 子目录 | 说明 |
|--------|------|
| `01_quickstart_constructor/` | 快速入门：构造器创建 |
| `02_turn_run/` | 运行单次对话轮次 |
| `03_turn_stream_events/` | 流式接收对话事件 |
| `04_models_and_metadata/` | 模型和元数据查询 |
| `05_existing_thread/` | 使用已有线程 |
| `06_thread_lifecycle_and_controls/` | 线程生命周期和控制 |
| `07_image_and_text/` | 图文混合输入 |
| `08_local_image_and_text/` | 本地图片和文本 |
| `09_async_parity/` | 异步对等测试 |
| `10_error_handling_and_retry/` | 错误处理和重试 |
| `11_cli_mini_app/` | CLI 迷你应用 |
| `12_turn_params_kitchen_sink/` | 对话参数全集演示 |
| `13_model_select_and_turn_params/` | 模型选择和对话参数 |
| `14_turn_controls/` | 对话控制功能 |
